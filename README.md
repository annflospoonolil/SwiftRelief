# SwiftRelief – Emergency Triage System

SwiftRelief is a full-stack web application designed to simulate an emergency triage system. It prioritizes patients based on symptoms and location, helping streamline emergency response.

---

## 🧠 Key Features

- 🔴 Classifies patients into priority levels (P1, P2, P3)
- 📍 Real-time location tracking using Browser Geolocation API
- ⚡ Fast and efficient backend using FastAPI
- 🌐 Interactive frontend built with React
- 🗄️ PostgreSQL database for structured data storage

---

## 💡 Problem Statement

In emergency situations, identifying critical cases quickly is crucial. SwiftRelief aims to assist by providing a system that:

- Evaluates symptoms
- Assigns urgency levels
- Helps prioritize response efficiently

---

## 🛠️ Tech Stack

**Frontend:**

- React.js
- HTML, CSS, JavaScript

**Backend:**

- FastAPI (Python)

**Database:**

- PostgreSQL

---

## ⚙️ How It Works

1. User enters symptoms
2. System processes input via backend API
3. Priority level is assigned:
   - P1 → Critical
   - P2 → Moderate
   - P3 → Low
4. Location is captured for better response planning
